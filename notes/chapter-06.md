## Урок 6: Визуальные эффекты и Декор

### 1. Теоретический минимум (Senior Notes)

**A. Тени (Shadows): Глубина вместо плоскости**
В реальном мире у объектов не бывает одной резкой тени. Свет падает с разных сторон.
Чтобы сделать «дорогую» тень, мы используем **многослойность**:
```css
/* Плохо (Junior): */
box-shadow: 5px 5px 10px rgba(0,0,0,0.5);

/* Хорошо (Senior - Layered Shadows): */
box-shadow: 
  0 1px 2px rgba(0,0,0,0.1),  /* Мелкая деталь */
  0 4px 8px rgba(0,0,0,0.1),  /* Средняя глубина */
  0 12px 24px rgba(0,0,0,0.1); /* Глубокая тень */
```
*Термин:* **Elevation** (Высота). Чем больше размытие и смещение, тем "выше" объект парит над фоном.

**B. Градиенты (Gradients)**
Градиент — это не цвет, это *картинка* (`background-image`).
*   **Linear:** `linear-gradient(45deg, #f00, #00f)` — классика.
*   **Radial:** `radial-gradient(circle at top right, ...)` — имитация источника света.
*   **Text Gradient:** Чтобы покрасить текст градиентом:
    ```css
    background: linear-gradient(...);
    background-clip: text;
    -webkit-background-clip: text;
    color: transparent;
    ```

**C. Фильтры и Стекло (Filters)**
*   `filter: blur(10px)` — размывает сам элемент.
*   `backdrop-filter: blur(10px)` — размывает то, что находится **ПОД** элементом (эффект матового стекла, как в iOS/macOS). **Важно:** требует полупрозрачного фона (`rgba` или `hsla`).

**D. Обрезка (Clipping)**
`clip-path` — это «ножницы» CSS. Ты можешь вырезать любую форму (треугольник, многоугольник) из квадратного `div`.
*   `clip-path: polygon(0 0, 100% 0, 100% 80%, 0 100%);` — скошенный низ.
