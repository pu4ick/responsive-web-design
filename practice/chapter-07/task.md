### 2. Практическое задание (Social Footer Component)

**Контекст:**
Ты верстаешь "подвал" сайта. Там есть блок с соцсетями (Facebook, Twitter, Instagram).
Дизайнер требует, чтобы иконки были серыми, но при наведении становились цветными (у каждой свой фирменный цвет).

**Задача:**
Реализовать систему иконок через **SVG Sprite** (технология `<symbol>`) и стилизовать их смену цвета через CSS.

**Технические требования:**

**1. HTML:**
*   В самом начале `<body>` создай скрытый контейнер SVG с шириной и высотой 0.
*   Внутри него используй теги `<symbol id="icon-name" viewBox="...">` для объявления иконок.
*   В самом футере используй конструкцию `<svg><use href="#icon-name"></use></svg>` для вставки.
*   *Данные для иконок (Paths):* Я дам тебе готовые пути ниже, чтобы ты не искал их.

**2. CSS:**
*   Задай базовый класс `.icon` (размер, базовый цвет заливки).
*   Используй `fill: currentColor`, чтобы иконка наследовала цвет текста ссылки.
*   При наведении на ссылку (`a:hover`) меняй `color` ссылки, и иконка должна автоматически перекраситься.
    *   Facebook: `#1877F2`
    *   Twitter (X): `#000000` (или темно-серый)
    *   Instagram: `#E4405F`
