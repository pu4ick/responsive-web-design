
### Code Review (Разбор полетов)
**1. Критическая ошибка: Структура списка (`li`)**
У тебя `
- ` (list item) вложены прямо в ``. Это **невалидный HTML**.
> **Правило:** Тег `
- ` может быть прямым потомком ТОЛЬКО тегов `
` (маркированный список) или `
` (нумерованный список).
Браузер попытается это исправить, но семантика будет сломана. Внутри `` всегда должна быть обертка из списка.
**2. Проблема доступности: Пустая ссылка (Logo)**
```html
```
Ссылка пустая. Скринридер (программа для незрячих) прочтет это как "Ссылка слэш" (читая href) или вообще проигнорирует.
> **Fix:** Добавь внутрь текст "Название Блога" или `img` с атрибутом `alt="Название Блога"`. Если картинка фоновая через CSS (что не рекомендуется для лого), нужен `aria-label`.
**3. Семантическая ошибка: `blockquote` для биографии**
Ты обернул текст "Об авторе" в `
` (цитата).
> **Вопрос:** Является ли биография автора цитатой? Нет. Это просто контент.
> **Fix:** Используй `
` или `
` внутри ``. `
` используем только если это прямая речь.
**4. Пропущенный элемент**
В задании требовался `` (подвал) с копирайтом. В коде его нет.
